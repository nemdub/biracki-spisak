Skripta vodi korisnika kroz proces odabira izbora, opštine/grada, biračkih mesta i unosa ličnih podataka (JMBG i broj lične karte), zatim učitava spisak glasača i snima ih u CSV fajlove.

Testirano na macOS.
Neophodni programi: `jq`, `curl`, `sed`, `grep`.

Za Windows koristite `.bat` ili direktno PowerShell (`.ps1`) fajl.
